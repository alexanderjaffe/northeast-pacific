# northeast-pacific

Code supporting "Deeply-sequenced metagenomes and over 1000 draft genomes from the epipelagic to bathypelagic Northeast Pacific Ocean" (Jaffe et al., 2026).

## code

*`snakemake/` - Snakemake workflow for viral identification and quantification.<br>
*`notebooks/` - Code describing the main analyses and figure generation are available here in notebook format.
   * `oc17-sra-compare-public.ipynb` - Perform analysis comparing sequencing effort with depth to public datasets.
   * `oc17-sd-refine.ipynb` - Perform manual genome curation for metagenomic bins, evaluate quality.
   * `oc17-sd-genomes-public.ipynb` - Analyze characteristics of draft genomes, including taxonomy, abundance, distribution, phylogenetic novelty.
   * `oc17-sd-binning-etc.ipynb` - Analyze community composition of metagenomes, binning/assembly effort.
